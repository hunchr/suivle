# Suivle
## Getting Started
1. Install [Bun](https://bun.sh/) if you haven't already
2. Clone the [boilerplate](https://github.com/hunchr/suivle-boilerplate)
3. Start the server
```bash
cd suivle-boilerplate
bun run dev
```
